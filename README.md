# Portfólio de Marcus Vinnicius
Este é o meu portfólio pessoal, hospedado no GitHub Pages. Ele apresenta minhas skills, experiências profissionais, projetos do GitHub e informações de contato. O design é inspirado em uma temática "Matrix" sutil, com foco em lógica de programação e vibe dev back-end.

## Tecnologias Usadas

HTML5
CSS3 (com animações e responsividade)
JavaScript (para efeitos dinâmicos e integração com GitHub API)
Canvas para o background animado

## Como Rodar Localmente

Clone o repositório: git clone https://github.com/omarcus212/portifolio.git
Abra o arquivo index.html em um navegador.

## Deploy no GitHub Pages

O site está deployado em: https://omarcus212.github.io/portifolio/
Para atualizar: Commit e push para o branch gh-pages ou configure nas settings do repo.

## Personalizações

Atualize o arquivo script.js com seu username do GitHub.
Substitua imagens e links no HTML conforme necessário.
